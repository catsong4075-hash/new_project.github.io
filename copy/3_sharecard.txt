✅ 3. 공유 카드 페이지 (share-card)

[카드 미리보기 영역 안내문]
카피 제목:
“민지님의 오늘의 행운지수를 카드로 만들었어요!”

설명 문구:
오늘의 점수와 행운 강화법이 담긴 나만의 카드예요.
친구들과 나눠보면 서로의 운도 더 좋아질지 몰라요 :)

[카드 구성 안내 문구]
포함 요소 소개:
📌 오늘의 점수 ·
📌 한 줄 운세 요약 ·
📌 실천 가능한 행운 강화법 리스트
예쁘고 감성적인 카드로 저장되었어요.

카드 내 텍스트 예시 요약 (실제 카드에 포함될 것처럼 제안):
“87점 – 민지님에게 좋은 기운이 가득해요!”
“초록색 옷과 가벼운 산책으로 오늘의 기운을 채워보세요.”

[공유 버튼 영역 안내]
저장하기 버튼 문구:
“이미지로 저장하기”

공유하기 버튼 문구:
“SNS에 공유하기”

스낵바 안내:
 - 저장 성공 시: “오늘의 행운지수를 저장했어요.”
 - 공유 클릭 시: “URL을 복사했어요.”

✴ 카피 설계 포인트

시선 흐름: “타이틀 → 설명 → 카드 미리보기 → 공유 유도”

감정 설계: 공감(행운 공유), 행동(카드 저장/공유) 중심

SNS 확산 전략: 감성형 문구 + 카드 콘텐츠 시각 강조
